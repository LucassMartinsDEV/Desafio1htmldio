Estrutura basica do HTML

voltar

O HTML possui uma estrutura fundamental bastante simples, sendo necessário o uso das tags <html> e </html>, <head> e </head>, <body> e <body>. Com essas instruções básicas, já é possível ter uma página web, mas podemos enriquecê-la ainda mais. É possível inserir diversas outras tags e aplicar estilos e comportamentos específicos em cada uma delas, de maneira individual ou mais abrangente.

Falando de tags

voltar

As tags são instruções que indicam ao navegador para executar uma função específica na estrutura de uma página web. Por exemplo, a tag <html> sinaliza o início do documento HTML.

Atributos das tags

voltar

Atributos são instruções que definem como as tags devem se comportar em uma página web. Os atributos podem ser globais, aplicáveis a várias tags, ou específicos para uma única tag. Por exemplo, o atributo id permite que definamos um nome único para um elemento na página.

Textos

voltar

A tipografia é algo fundamental no design web, pois com ela podemos exibir textos em vários formatos em nosso site. Mas não é apenas isso; com a tipografia, podemos criar uma estética visualmente bonita para nossa página web. Além disso, as tags de cabeçalho de <h1> a <h6> nos ajudam a estabelecer uma hierarquia entre os elementos do cabeçalho

Listas Ordenadas e Não ordenadas

voltar

Listas ordenadas e não ordenadas são formas de apresentar conteúdo em uma página ou documento. Enquanto a lista ordenada (<ol>) organiza os itens de forma sequencial, a lista não ordenada (<ul>) utiliza marcadores antes de cada item para destacá-los sem uma ordem específica.

Links

voltar

Links, ou hiperlinks, são um dos conceitos mais fundamentais da web. Eles permitem que os usuários naveguem de uma página para outra, conectem-se a diferentes seções dentro de uma mesma página, baixem arquivos, enviem emails e muito mais. No HTML, a tag <a>, também conhecida como âncora, é usada para criar links.
